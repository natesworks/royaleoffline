# NBS Offline

An offline client for newer versions of the game. It supports v59 (Android and iOS) and v64 (Android only for now).

## Download

v59: [APK](https://brawlmods.natesworks.com/mod?id=nbs) [iPA](https://brawlmods.natesworks.com/mod?id=nbs)

Script downloads available in the [releases section](https://github.com/nbs-offline/nbs-offline/releases).

## Features

- Main menu
- Offline battles
- Extensive configuration
- Player profile
- Changing name
- Saving thumbnail, name color, favourite brawler and more
- Random bot names
- Content creator codes
- Teams (wip)
- Starr road (wip)
- Tutorial
- Better loading screen
- Minimal settings
